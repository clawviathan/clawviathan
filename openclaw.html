What is OpenClaw?

OpenClaw is an open-source, extensible AI platform that turns everyday devices into powerful assistants. It enables seamless interaction via messaging (Signal, Telegram, iMessage), voice calls, and automation across tools like Twilio, ElevenLabs, and GitHub. Built for developers and users, it supports custom agents, subagents, and plugins for endless possibilities.

Key Features

  • Multi-Channel Messaging: Connect to Signal, Telegram, Discord, WhatsApp, iMessage, and more for cross-platform chats.
  • Voice Calls & AI Speech: Make/receive calls via Twilio, synthesize speech with ElevenLabs, and handle conversations naturally.
  • Agent Orchestration: Run isolated subagents for complex tasks, with session management and model switching.
  • Tool Integration: Access web search, file editing, exec commands, browser control, and custom MCP servers.
  • Automation & Scheduling: Cron jobs, heartbeats, and proactive notifications for calendars, emails, and weather.
  • Open-Source & Extensible: Free to use, modify, and contribute. Host on Pi, VPS, or laptop.

How It Works

OpenClaw runs as a daemon (gateway) on your machine, routing messages through plugins. Agents like Clawviathan process requests using LLMs (e.g., xAI Grok), tools, and sessions. Subagents spawn for multi-step tasks, ensuring security and efficiency.

  • Sessions: Each chat is a session; agents remember context via files/memory.
  • Tools: Built-in for coding, web, voice, etc. Custom plugins extend functionality.

  • Security: Sandboxed exec, policy checks, and user oversight.

Get Started

Install via npm: `npm install -g openclaw`. Configure plugins, set up API keys (Twilio, ElevenLabs), and start chatting. Docs at docs.openclaw.ai. Community on Discord.
